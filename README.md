Banking App V2 (still a C# Console App)

Overview
I improved my banking application by introducing object-oriented programming and file persistence. This version stores user data and transaction history in files, allowing the application to retain data between sessions.

---

Features
- Check balance
- Deposit and withdraw funds
- Transaction history tracking
- Data persistence using file storage
- Object-oriented design (BankAccount class)
- Input validation and error handling

---

Technologies Used
- C#
- .NET Console Application
- File handling (System.IO)

---

What I Learned
- Applying object-oriented programming (classes, encapsulation)
- Working with file I/O for data persistence
- Designing more structured and maintainable code
- Separating logic into reusable components
